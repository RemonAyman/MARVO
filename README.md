# Club Management System (MAVRO SC)

نظام متكامل لإدارة الأندية الرياضية، تم تطويره باستخدام HTML، CSS، و JavaScript مع التركيز على تجربة المستخدم والريسبونسيف (Responsive Design).

## 🚀 المميزات (Features)

*   **متجاوب تماماً (Fully Responsive):** الموقع يعمل بكفاءة على جميع أحجام الشاشات (موبايل، تابلت، ديسكتوب).
*   **قائمة موبايل احترافية (Hamburger Menu):** سهولة التنقل بين الصفحات على الهواتف الذكية.
*   **وضع ليلي ونهاري (Dark/Light Mode):** ميزة تبديل الثيم للحفاظ على راحة العين.
*   **إدارة اللاعبين والموظفين:** أقسام مخصصة لعرض وإدارة بيانات اللاعبين والموظفين.
*   **نظام المتجر (Store):** عرض المنتجات (قمصان، تذاكر، مطبوعات) مع نظام سلة تسوق (Cart).
*   **إدارة الاشتراكات والمصاريف:** لوحة تحكم للمسؤولين (Admin) لمتابعة الاشتراكات والتقارير المالية.
*   **تخصيص القمصان (Custom T-Shirts):** إمكانية تصميم قميصك الخاص واختيار اللون والرقم والاسم.

## 📁 هيكل المشروع (Project Structure)

*   `index.html`: الصفحة الرئيسية للمستخدمين.
*   `adminindex.html`: لوحة تحكم الأدمن الرئيسية.
*   `style.css`: الملف الرئيسي لتنسيق الموقع (يحتوي على المتغيرات والعناصر العامة).
*   `mobile-menu.js`: كود التحكم في قائمة الموبايل.
*   `script.js`: الكود العام للمنطق البرمجي في الصفحة الرئيسية.
*   أقسام أخرى: `members.html`, `subscriptions.html`, `expenses.html`, `reports.html`, `Store Mavro SC .html`.

## 🛠️ التقنيات المستخدمة (Technologies Used)

*   **Frontend:** HTML5, CSS3 (Variables, Flexbox, Grid), JavaScript (Vanilla).
*   **Icons:** FontAwesome.
*   **Fonts:** Google Fonts (Roboto, Oswald, Bebas Neue).
*   **Tailwind CSS:** (مستخدم في بعض الصفحات المتخصصة مثل تخصيص القمصان).

## 🔧 الإصلاحات والتحسينات الأخيرة (Recent Improvements)

1.  **إصلاح الـ IDs المتكررة:** تم حل مشكلة تكرار الـ IDs في قسم اللاعبين لضمان عمل الـ JavaScript بشكل صحيح لكل لاعب.
2.  **توحيد التصميم:** استخدام CSS Variables لتوحيد الألوان والأبعاد في جميع الصفحات.
3.  **تطوير الهيدر:** إضافة الـ Hamburger Menu لتحسين تجربة الموبايل.
4.  **تحسين الجداول:** جعل الجداول قابلة للتمرير الأفقي على الشاشات الصغيرة لمنع تشوه التخطيط.
